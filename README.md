# 🏅 Olympics Data Analysis Dashboard

An interactive data analysis web app built using **Streamlit** to explore historical Olympic data.

## 🚀 Live Demo

👉 https://olympics-dashboard-zh7u.onrender.com/
---

## 📊 Features

* 🥇 **Medal Tally Analysis**
  View country-wise medal distribution across years

* 🌍 **Overall Analysis**
  Insights into participation, events, and trends

* 🗺️ **Country-wise Analysis**
  Performance of individual countries over time

* 🧑‍🤝‍🧑 **Athlete Analysis**
  Age distribution, gender participation, and medal trends

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit
* Pandas
* NumPy
* Matplotlib / Seaborn
* Plotly

---

## 📁 Dataset

* `athlete_events.csv`
* `noc_regions.csv`

---

## ⚙️ Installation (Local)

```bash
git clone https://github.com/your-username/olympics-dashboard.git
cd olympics-dashboard
pip install -r requirements.txt
streamlit run app.py
```
